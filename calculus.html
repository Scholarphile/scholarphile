Calculus I & II

Master differential and integral calculus from limits and derivatives to advanced integration techniques and infinite series. Essential for all STEM fields.

Chapter 1: Limits and Continuity

1.1 Introduction to Limits

The limit of a function $f(x)$ as $x$ approaches $a$ is the value that $f(x)$ gets arbitrarily close to:

$$\lim_{x \to a} f(x) = L$$

This means: for every $\epsilon > 0$, there exists $\delta > 0$ such that if $0 < |x - a| < \delta$, then $|f(x) - L| < \epsilon$.

Example: Basic Limit

Find $\lim_{x \to 2} (3x + 1)$

$$\lim_{x \to 2} (3x + 1) = 3(2) + 1 = 7$$

For polynomial functions, we can substitute directly.

1.2 Limit Laws

If $\lim_{x \to a} f(x) = L$ and $\lim_{x \to a} g(x) = M$, then:

  • Sum Rule: $\lim_{x \to a} [f(x) + g(x)] = L + M$
  • Product Rule: $\lim_{x \to a} [f(x) \cdot g(x)] = L \cdot M$
  • Quotient Rule: $\lim_{x \to a} \frac{f(x)}{g(x)} = \frac{L}{M}$ (if $M \neq 0$)
  • Power Rule: $\lim_{x \to a} [f(x)]^n = L^n$

1.3 Indeterminate Forms and L'Hôpital's Rule

When we encounter $\frac{0}{0}$ or $\frac{\infty}{\infty}$, we can use L'Hôpital's Rule:

$$\lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)}$$

Example: L'Hôpital's Rule

Find $\lim_{x \to 0} \frac{\sin x}{x}$

Direct substitution gives $\frac{0}{0}$, so we apply L'Hôpital's Rule:

$$\lim_{x \to 0} \frac{\sin x}{x} = \lim_{x \to 0} \frac{\cos x}{1} = \frac{\cos 0}{1} = 1$$

1.4 Continuity

A function $f(x)$ is continuous at $x = a$ if:

  1. $f(a)$ exists
  2. $\lim_{x \to a} f(x)$ exists
  3. $\lim_{x \to a} f(x) = f(a)$

Chapter 2: Derivatives

2.1 Definition of the Derivative

The derivative of $f(x)$ at $x = a$ is:

$$f'(a) = \lim_{h \to 0} \frac{f(a+h) - f(a)}{h}$$

Geometrically, this represents the slope of the tangent line at point $(a, f(a))$.

Example: Derivative from Definition

Find the derivative of $f(x) = x^2$ using the definition:

$$f'(x) = \lim_{h \to 0} \frac{(x+h)^2 - x^2}{h}$$
$$= \lim_{h \to 0} \frac{x^2 + 2xh + h^2 - x^2}{h} = \lim_{h \to 0} \frac{2xh + h^2}{h}$$
$$= \lim_{h \to 0} (2x + h) = 2x$$

2.2 Basic Differentiation Rules

  • Power Rule: $\frac{d}{dx}[x^n] = nx^{n-1}$
  • Constant Rule: $\frac{d}{dx}[c] = 0$
  • Sum Rule: $\frac{d}{dx}[f(x) + g(x)] = f'(x) + g'(x)$
  • Product Rule: $\frac{d}{dx}[f(x)g(x)] = f'(x)g(x) + f(x)g'(x)$
  • Quotient Rule: $\frac{d}{dx}\left[\frac{f(x)}{g(x)}\right] = \frac{f'(x)g(x) - f(x)g'(x)}{[g(x)]^2}$

2.3 Chain Rule

For composite functions $f(g(x))$:

$$\frac{d}{dx}[f(g(x))] = f'(g(x)) \cdot g'(x)$$

Example: Chain Rule

Find $\frac{d}{dx}[\sin(x^2 + 1)]$

Let $f(u) = \sin u$ and $g(x) = x^2 + 1$, so $f'(u) = \cos u$ and $g'(x) = 2x$

$$\frac{d}{dx}[\sin(x^2 + 1)] = \cos(x^2 + 1) \cdot 2x = 2x\cos(x^2 + 1)$$

2.4 Derivatives of Common Functions

$$\frac{d}{dx}[\sin x] = \cos x \quad \frac{d}{dx}[\cos x] = -\sin x$$ $$\frac{d}{dx}[\tan x] = \sec^2 x \quad \frac{d}{dx}[e^x] = e^x$$ $$\frac{d}{dx}[\ln x] = \frac{1}{x} \quad \frac{d}{dx}[a^x] = a^x \ln a$$

2.5 Implicit Differentiation

For equations where $y$ is defined implicitly, differentiate both sides with respect to $x$:

Example: Implicit Differentiation

Find $\frac{dy}{dx}$ for $x^2 + y^2 = 25$

Differentiating both sides:

$$2x + 2y\frac{dy}{dx} = 0$$
$$\frac{dy}{dx} = -\frac{x}{y}$$

Chapter 3: Applications of Derivatives

3.1 Related Rates

When quantities are related by an equation, their rates of change are also related.

Example: Balloon Problem

A spherical balloon is being inflated. If the radius increases at 2 cm/min, how fast is the volume increasing when $r = 5$ cm?

Given: $\frac{dr}{dt} = 2$ cm/min, find $\frac{dV}{dt}$ when $r = 5$ cm

Volume of sphere: $V = \frac{4}{3}\pi r^3$

$$\frac{dV}{dt} = \frac{dV}{dr} \cdot \frac{dr}{dt} = 4\pi r^2 \cdot 2 = 8\pi r^2$$

When $r = 5$: $\frac{dV}{dt} = 8\pi(25) = 200\pi$ cm³/min

3.2 Optimization

To find maximum and minimum values:

  1. Find critical points: $f'(x) = 0$ or $f'(x)$ undefined
  2. Use the second derivative test: $f''(x) > 0$ (minimum), $f''(x) < 0$ (maximum)
  3. Check endpoints if on a closed interval

Example: Optimization

Find the dimensions of a rectangle with perimeter 20 that has maximum area.

Let length = $x$ and width = $y$. Then $2x + 2y = 20$, so $y = 10 - x$

Area: $A = xy = x(10-x) = 10x - x^2$

$$A'(x) = 10 - 2x = 0 \Rightarrow x = 5$$
$$A''(x) = -2 < 0$$

So $x = 5$, $y = 5$. The rectangle is actually a square!

3.3 Mean Value Theorem

If $f$ is continuous on $[a,b]$ and differentiable on $(a,b)$, then there exists $c \in (a,b)$ such that:

$$f'(c) = \frac{f(b) - f(a)}{b - a}$$

Chapter 4: Integration

4.1 Antiderivatives

An antiderivative of $f(x)$ is a function $F(x)$ such that $F'(x) = f(x)$.

The indefinite integral: $\int f(x) \, dx = F(x) + C$

4.2 Basic Integration Rules

$$\int x^n \, dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$$ $$\int \frac{1}{x} \, dx = \ln|x| + C$$ $$\int e^x \, dx = e^x + C$$ $$\int \sin x \, dx = -\cos x + C$$ $$\int \cos x \, dx = \sin x + C$$

4.3 Substitution Method

For integrals of the form $\int f(g(x))g'(x) \, dx$, use substitution $u = g(x)$:

Example: U-Substitution

Evaluate $\int 2x(x^2 + 1)^3 \, dx$

Let $u = x^2 + 1$, then $du = 2x \, dx$

$$\int 2x(x^2 + 1)^3 \, dx = \int u^3 \, du = \frac{u^4}{4} + C = \frac{(x^2 + 1)^4}{4} + C$$

4.4 Integration by Parts

For products of functions: $\int u \, dv = uv - \int v \, du$

Example: Integration by Parts

Evaluate $\int x e^x \, dx$

Let $u = x$ (so $du = dx$) and $dv = e^x dx$ (so $v = e^x$)

$$\int x e^x \, dx = x e^x - \int e^x \, dx = x e^x - e^x + C = e^x(x-1) + C$$

4.5 Definite Integrals

The Fundamental Theorem of Calculus:

$$\int_a^b f(x) \, dx = F(b) - F(a)$$

where $F'(x) = f(x)$.

Example: Definite Integral

Evaluate $\int_0^2 x^2 \, dx$

$$\int_0^2 x^2 \, dx = \left[\frac{x^3}{3}\right]_0^2 = \frac{8}{3} - 0 = \frac{8}{3}$$

Chapter 5: Applications of Integration

5.1 Area Between Curves

The area between curves $y = f(x)$ and $y = g(x)$ from $x = a$ to $x = b$:

$$A = \int_a^b |f(x) - g(x)| \, dx$$

Example: Area Between Curves

Find the area between $y = x^2$ and $y = 2x$ from $x = 0$ to $x = 2$

First, find intersection points: $x^2 = 2x \Rightarrow x = 0, 2$

Since $2x > x^2$ on $(0,2)$:

$$A = \int_0^2 (2x - x^2) \, dx = \left[x^2 - \frac{x^3}{3}\right]_0^2 = 4 - \frac{8}{3} = \frac{4}{3}$$

5.2 Volumes of Solids of Revolution

Disk Method

Volume when rotating $y = f(x)$ around the x-axis:

$$V = \pi \int_a^b [f(x)]^2 \, dx$$

Shell Method

Volume when rotating around a vertical line:

$$V = 2\pi \int_a^b x \cdot f(x) \, dx$$

Example: Volume of Revolution

Find the volume when $y = x^2$ from $x = 0$ to $x = 1$ is rotated around the x-axis.

$$V = \pi \int_0^1 (x^2)^2 \, dx = \pi \int_0^1 x^4 \, dx = \pi \left[\frac{x^5}{5}\right]_0^1 = \frac{\pi}{5}$$

5.3 Arc Length

The arc length of $y = f(x)$ from $x = a$ to $x = b$:

$$L = \int_a^b \sqrt{1 + [f'(x)]^2} \, dx$$

5.4 Work and Average Value

Work: $W = \int_a^b F(x) \, dx$

Average Value: $f_{avg} = \frac{1}{b-a} \int_a^b f(x) \, dx$

Chapter 6: Infinite Series

6.1 Sequences and Series

A sequence is a list of numbers: $\{a_n\} = a_1, a_2, a_3, \ldots$

A series is the sum of a sequence: $\sum_{n=1}^{\infty} a_n = a_1 + a_2 + a_3 + \cdots$

6.2 Convergence Tests

Geometric Series

$$\sum_{n=0}^{\infty} ar^n = \frac{a}{1-r} \quad \text{if } |r| < 1$$

Ratio Test

For $\sum a_n$, if $\lim_{n \to \infty} \left|\frac{a_{n+1}}{a_n}\right| = L$:

  • If $L < 1$, the series converges
  • If $L > 1$, the series diverges
  • If $L = 1$, the test is inconclusive

6.3 Power Series

A power series has the form: $\sum_{n=0}^{\infty} c_n(x-a)^n$

Every power series has a radius of convergence $R$ such that:

  • The series converges if $|x-a| < R$
  • The series diverges if $|x-a| > R$

6.4 Taylor and Maclaurin Series

The Taylor series of $f(x)$ centered at $x = a$:

$$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(x-a)^n$$

Common Maclaurin series (centered at $x = 0$):

$$e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!} = 1 + x + \frac{x^2}{2!} + \frac{x^3}{3!} + \cdots$$
$$\sin x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n+1}}{(2n+1)!} = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots$$
$$\cos x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n}}{(2n)!} = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \cdots$$

Chapter 7: Real-World Applications

7.1 Physics Applications

  • Velocity and Acceleration: $v(t) = \frac{ds}{dt}$, $a(t) = \frac{dv}{dt}$
  • Work: $W = \int F(x) \, dx$
  • Center of Mass: $\bar{x} = \frac{\int x \, dm}{\int dm}$

7.2 Economics Applications

  • Marginal Cost: $MC = \frac{dC}{dx}$
  • Consumer Surplus: $CS = \int_0^{x_0} [D(x) - p_0] \, dx$
  • Present Value: $PV = \int_0^T R(t)e^{-rt} \, dt$

7.3 Biology Applications

  • Population Growth: $\frac{dP}{dt} = kP$ (exponential model)
  • Logistic Growth: $\frac{dP}{dt} = kP(1 - \frac{P}{M})$
  • Drug Concentration: Exponential decay models
